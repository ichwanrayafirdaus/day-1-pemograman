# 🛒 Toko Kita

![PWA](https://img.shields.io/badge/PWA-Ready-4F46E5?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Active-success?style=for-the-badge)

Toko Kita adalah aplikasi Progressive Web App (PWA) sederhana untuk membantu mengelola keuangan bisnis seperti mencatat pemasukan dan pengeluaran dengan mudah, cepat, dan praktis.

---

## 📸 Preview Aplikasi
> Tambahkan screenshot aplikasi di sini (opsional)

---

## 🚀 Fitur Utama
- 💰 Mencatat pemasukan & pengeluaran
- 📊 Menampilkan total transaksi
- 💾 Penyimpanan otomatis (localStorage)
- 📱 Bisa di-install seperti aplikasi (PWA)
- ⚡ Bisa digunakan secara offline
- 🎨 Tampilan responsif dan modern

---

## 🧠 Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript
- Service Worker (PWA)
- Web App Manifest

---

## 📂 Struktur Project
toko-kita/ ├── index.html ├── manifest.json ├── service-worker.js ├── offline.html ├── assets/ │   ├── style.css │   └── icons/ ├── README.md

---

## 🛠️ Cara Menjalankan
1. Download / clone project ini
2. Buka folder project
3. Jalankan dengan Live Server atau buka `index.html`

---

## 📲 Cara Install ke HP
1. Buka di browser Chrome
2. Klik menu (⋮)
3. Pilih **"Tambahkan ke layar utama"**
4. Selesai 🎉

---

## 🎯 Tujuan Project
Aplikasi ini dibuat untuk membantu pelaku usaha kecil dalam mengelola keuangan secara sederhana dan efisien.
